# Intent: Agent Swarms & Orchestration (SYS-AGENTS)

## 1. Goals
Scale research and writing by delegating work to autonomous, stateful agent networks that audit and improve their own output.

## 2. Constraints
- Strict cost budgets per content generation run.
- Prevent infinite feedback loops in agent state transitions.

## 3. Success Criteria
- Research packs and scripts pass fact-check validation with a quality score above 90 without human intervention.
